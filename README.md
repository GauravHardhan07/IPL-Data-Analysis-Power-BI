# IPL-Data-Analysis-Power-BI
This Power BI project analyzes Indian Premier League (IPL) data from 2008 to 2025, providing season‑wise insights into team performance, player statistics, and match trends. The dashboard enables users to explore historical and current IPL seasons interactively.

# Insights
Automated season‑wise dashboards reduce manual reporting effort.
Data model supports advanced KPIs such as total runs, wickets, and boundary counts.
Enables quick comparison of team and player performance across years.
